Switch statements are useful for when checking a single variable's value and doing different things depending on that value.

```javascript
switch (thing) {
  case 'stuff':
    console.log('Stuff!');
    break;
  case 'cheese':
    console.log('Cheese!');
    break;
  case 'fluff':
    console.log('Fluff!');
    break;
  default:
    console.log('Help!');
    break;
}
```
